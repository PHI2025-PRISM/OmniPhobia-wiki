# 病历编号：OP-1000

**疾病名称：蜈蚣恐惧症**

**危险等级：IV级**

**病号描述：**

OP-1000是一个脸色憔悴，24小时身穿密封紧身防护服的男子，高1米72，重40千克，被收容在一间24小时灯光照耀，没有一丝阴影，角落和缝隙的双层房间里，房间里没有任何能映照相貌的事物，房间里放着旋律不同，没有较长重复片段的音乐。

OP-1000会对一切阴暗角落和称得上缝隙孔洞的地方恐惧，这种恐惧并非是对阴暗和缝隙的恐惧，而是对阴暗缝隙中可能存在的蜈蚣的恐惧，坚信任何一处阴暗角落和缝隙都有可能存在蜈蚣，并且OP-1000对细微和连续相同的声音敏感，坚信那一定是蜈蚣爬行的声音。

**_发病症状：_**

病号的发病症状有四种，分别是OP-1000-1，OP-1000-2，OP-1000-3和OP-1000-4。

**_发病症状OP-1000-1_**

OP-1000会不自觉的抓挠自身，随后以OP-1000为中心，半径为3公里的圆内，一切算得上洞隙的阴暗地方，都会钻出大量的蜈蚣，比如人的毛孔和嘴耳鼻，排泄口，隧道等等，蜈蚣的大小取决于洞的大小，无论长度大小如何，蜈蚣的关节和脚都有千数之多，该现象似乎与OP-1000认知中的蜈蚣为千节千足之虫有关。OP-1000-1的持续时间随机，基本在停止抓挠后结束发病。

OP-1000-1期间，体内冒出蜈蚣的受害者并不会立刻死亡，而是不停冒出蜈蚣，同时受害者的感官变得异常敏感，无时无刻经历着蜈蚣在身上不断爬行和行走时细微摩擦声的折磨，在OP-1000-1结束后，哪怕受害者经过系统性的专业治疗，也会留有严重的后遗症，比如对一切声音敏感和不自觉的抓挠自身。

**（注：OP-1000认知中的阴暗角落是指人无法看到内部的地方，别人看不到也算看不到，所以蜈蚣会在任何角落出现）**

**_发病症状OP-1000-2_**

OP-1000会被变得异常安静，不会被任何外界因素影响，同时以OP-1000为中心半径两公里的圆内，一切活物的孔洞和缝隙都会被填充粘合，眼皮被粘合，毛孔被填充，身体变得光滑，意识和感知消失，转化成只是活着但没有意识的光滑肉团，且转化过程不可逆，在经过一段时间后，光滑肉团会直接失去生命，变成坚硬光滑的固体物。

**（注：失去生命的时间长短取决于活物体型的大小）**

**_发病症状OP-1000-3_**

OP-1000会在地上扭曲蠕动爬行，同时以OP-1000为中心半径四公里的圆内，所有动物的关节和脊椎被异化成蜈蚣爬出，所有生物从一个个体直接被拆成数个个体，异化的蜈蚣还会保留一部分生物的特征，比如说千足是肋骨或者手指的模样。墓地里还没有分解掉的骨头，也会变成异化的骨头蜈蚣。

**（注：异化的蜈蚣还会保留一部分生物的意识）**

**发病症状OP-1000-4**

OP-1000会突然把头伸到屁股那里，以一种奇怪的姿势滚动，同时以OP-1000为中心半径3.5公里的圆内，所有生物都以头尾串联的方式组成怪诞的人体蜈蚣，消失的头部会在人体蜈蚣的体表哀嚎蠕动，并发出连续密集的刺耳声音。

**附录：**

**附录1：对OP-1000的实验**

在一次实验中，研究人员尝试封闭OP-1000的视觉，在封闭成功时，OP-1000突然表现出了极度的恐惧，并开始发病，根据事后调查研究，判断OP-1000在封闭视觉后，可能陷入一片黑暗的状态，导致引起了发病。

而在另一次实验中，研究人员封闭了OP-1000的听觉，前一段时间OP-1000表现正常，但过一段时间后，OP-1000产生了严重的幻听，并开始发病。

后续的屏蔽感官实验，因为前两次事故的发生，科研人员直接强行麻痹和切断OP-1000的感知，使其处于活死人的状态，实验进行的很顺利，但过了一段时间好后，第一次OP-1000-2病症发生，后使用机器人重新激活OP-1000的感官后才结束这场可怕的病症。
